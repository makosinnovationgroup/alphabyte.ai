# Resolve brand direction: 2021 guide aesthetic vs. Ventures aesthetic

## Type
Decision

## Status
Resolved

## Context
alphabyte.ai needs a settled visual language before any major UI component ships. There are two competing aesthetics:

1. **2021 brand guide** — light, clean, Alphabyte Blue dominant on white surfaces. This is what the current brand skill files codify.
2. **alphabyte.com Ventures site** — dark backgrounds, video heroes, circle/orb motifs. More dramatic and moody.

The header, footer, hero, and every surface-level color decision depends on which direction wins. Components built against the wrong aesthetic will need rework.

## Desired outcome

- If it's a hybrid (e.g., mostly 2021 guide with select dark sections), define exactly which pages or sections get which treatment
- Update brand skill files if any rules change as a result
