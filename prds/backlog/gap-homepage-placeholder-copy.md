# Replace homepage placeholder body copy that violates brand voice guidelines

## Type
Gap

## Status
Promoted

## Context
The homepage currently displays placeholder body copy lifted from the 2021 brand narrative: "Alphabyte applies deep technological expertise, data science methodologies and specialised know-how to ignite meaningful change for our clients." The word "ignite" appears in the voice-and-tone.md forbidden vocabulary section, and the overall phrase structure is hype-adjacent and corporate-mystical — both patterns the brand voice guidelines explicitly prohibit. This copy will be replaced when the homepage PRD is written via `/feature`, so the issue is expected to self-resolve.
